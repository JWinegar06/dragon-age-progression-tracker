# Dragon Age Progression Tracker

A simple progression tracker built with Next.js, Tailwind CSS, Firebase, and React Hook Form.

## Features

- Firebase email/password sign up and login
- Firebase Google sign-in with popup auth
- Firestore-backed progress entries
- Filters for platform, genre, status, and favorites
- Dark mode toggle
- Mobile-friendly layout

## Getting Started

1. Install dependencies:
npm install firebase react-hook-form clsx
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
npm run dev
